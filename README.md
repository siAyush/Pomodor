## About

Spotify Clone

## Setup Locally

```
$ git clone https://github.com/siAyush/Spotify-Clone.git
$ cd Spotify-Clone
$ npm install
$ npm run dev
```

## Dependencies

- Next.js
- Tailwind CSS
- Heroicons
- NextAuth.js
- Recoil

## Environment Variables
- Create .env file
-  Add this values
```
NEXTAUTH_URL = http://localhost:3000
NEXT_PUBLIC_CLIENT_SECRET = ''
NEXT_PUBLIC_CLIENT_ID = ''
JWT_SECRET = secret_value
```
